Frequently asked questions

What is included in the template?

The template includes over 100 components across categories like buttons, cards, headers, footers, forms, alerts, and more. Each component comes in multiple variants to fit your branding.

Do you offer support?

Yes, we provide community support via GitHub Issues and Discussions. For priority support, check our pricing plans which include dedicated assistance.

How often is the template updated?

We release updates regularly with new components, improvements, and bug fixes. All updates are seamlessly tested and made available for existing users.

What technologies does this template use?

This template is built with HTML, TailwindCSS, and vanilla JavaScript. It uses Vite as the build tool for fast development and optimized production builds.

Can I use this for commercial projects?

Yes! This template is available under a flexible license. You can use it freely for both personal and commercial projects without any major restrictions on the standard plans.

What browsers are supported?

All modern browsers are supported, including Chrome, Firefox, Safari, and Edge. We use progressive enhancement to ensure broad compatibility.
